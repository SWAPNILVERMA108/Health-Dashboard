## 🌟 Project Overview

🚀 A modern **Healthcare Dashboard Web App** that integrates:

* 📊 Real-time vital monitoring
* 💊 Digital prescriptions
* ⚠️ Critical health alerts
* 👨‍⚕️ Role-based access system

---

## 🎯 Features

✨ **Core Functionalities**

* ❤️ Live Heart Rate Monitoring
* 🩸 Blood Pressure Tracking
* 💧 SpO2 Monitoring
* 🌡️ Temperature & Respiration
* 📋 Patient History Management
* 💊 Digital Prescription System
* ⚠️ Emergency Alert System
